Area Of trapezium

Here, First Base, Second Base And Height Of trapezium Is Given And The Output Is Area Of trapezium. 
Java Code 

import java.util.Scanner;
public class Main{
    public static float Area_of_Trapezium(int First_Base, int Second_Base, int Height){
        float pi = 22f/7;
        float Area = ((First_Base + (float)Second_Base)/2)*Height; //  ((a+b)/2)*h
        return Area;
    }
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter First Base of trapezium:");
        int First_Base = sc.nextInt();
        System.out.print("Enter Second Base of trapezium:");
        int Second_Base = sc.nextInt();
        System.out.print("Enter Height of trapezium:");
        int Height = sc.nextInt();
        System.out.println("Area :"+ Area_of_Trapezium(First_Base, Second_Base, Height));
    }
}
